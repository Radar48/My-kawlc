# 🎀 My Pink Calculator

A cute and fully functional Android calculator app built using **Kotlin** and **XML layouts**.

---

## ✨ Features

* Basic arithmetic operations: ➕ ➖ ✖️ ➗
* Decimal number support
* Clear (reset) functionality
* Smart expression evaluation
* Emoji-style operator buttons 🎀
* Clean and minimal pink-themed UI

* **Language:** Kotlin
* **UI:** XML Layouts
* **IDE:** Android Studio
* **Library:** exp4j (for expression evaluation)

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Radar48/My-kawlc.git
```

### 2. Open in Android Studio

* Open Android Studio
* Click **Open Project**
* Select the project folder

### 3. Run the app

* Connect an emulator or Android device
* Click **Run ▶️**
